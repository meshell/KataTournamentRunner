    Kata Tournament Runner

    A application to run a Kyokushin Karate Kata Tournament according to the IFK Switzerland rules.

    Dependencies

    Building the "Kata Tournament Runner" requires a recent C++11 compliant compiler (e.g. gcc 4.7), Qt Version 5.2 and CMake (http://www.cmake.org/) Version 2.8 or higher. See CMake documentation for more information about building using CMake.

    Building the Unittests requires google mock (http://code.google.com/p/googlemock/) and google test (http://code.google.com/p/googletest/).

    Building the feature tests (BDD style tests using Gherkin) requires Cucumber-cpp (https://github.com/cucumber/cucumber-cpp) by Paolo Ambrosio, google test and Boost (regex, filesystem, date_time, chrono, thread, system).
    See the Cucumber-cpp documentation on how to run the feature tests with cucumber.

    Google mock, google test, cucumber-cpp and Boost can be downloaded and built by runing make prepare.
